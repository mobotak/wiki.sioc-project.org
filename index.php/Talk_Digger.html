About

Figure 2: Talk Digger schemas [1]

Talk Digger is a web service that help people to find, follow and enter conversations of the Web. Basically it is a service that tell to people who is linking to a specific web page. Users can create a personal profile, explicit their interests, make new friends, track conversations, leave comments in conversations, etc.

Considering all these features, this is the perfect match to use the SIOC ontology to export its data in RDF/XML.

The implementation strategy can be found in this blog post. All the mapping between Talk Digger's concepts and SIOC classes and properties are explained in that document.
