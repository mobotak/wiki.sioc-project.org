Standard autodiscovery pattern

Other autodiscovery patterns

TimBL

Meta link, relative URL, type = "application/rdf+xml", title = "Contact" 

 {{{<link rel="meta" type="application/rdf+xml" title="Contact" href="card.rdf" />}}}

Also used to have same link with title = "FOAF", disabled now. 

 add dates when added and when disabled.

Currently icon is not displayed because 'title' attribute is different. 

 Q: is it detected and pinged by SemRadar?

Knobot @ Danny Ayers

Alternate link, absolute URL, type = "application/rdf+xml", title = None 

 {{{<link href="http://dannyayers.com/.rdf?replaceLang=de&replaceLang=it&replaceLang=fr&replaceLang=en&amp;" title="RDF" type="application/rdf+xml" rel="alternate"/>}}}

Present on the main page, but not on pages with individual posts. 

 Note: RDF for individual pages can be retrieved by appending '.rdf' to the URL

Currently icon is not disaplayed because there is no title.

Q: is it detected and pinged by SemRadar?
