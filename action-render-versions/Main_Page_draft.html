This is the wiki of SIOC (Semantically Interlinked Online Communities) - an ontology for describing discussion forums and posts on topic threads in online community sites. This includes but is not limited to: blogs, bulletin boards, mailing lists, newsgroups, etc. The main SIOC project page serves as a repository for more permanent information and latest news.

In this wiki we keep information, ToDo lists and issues of existing applications using SIOC and work in progress.

You can also find guidelines on how to use SIOC, thoughts and ideas for its future and some geral semantic web related information.
