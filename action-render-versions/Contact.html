IRC

A quick way to get in touch with SIOCers is the #sioc irc chat channel on the irc.freenode.net server.

If you don't have an irc client or a irc enabled browser like Firefox with the Chatzilla add-on or Opera, then you can use this java applet. Just type 

/nick your_nickname
/join #sioc

It might talk a while until somebody responds, but we usually do!
