About

OpenLink Data Spaces is a Distributed Collaborative Application Suite built atop the OpenLink Virtuoso Universal Server platform.

It exposes SIOC instance data in the form of Real and Virtual RDF Graphs for the following applications realms: Weblog, Wiki, Feed Aggregation, Shared Bookmarks, Photo Gallery, Disucssion Forums, and WebDAV based File Servers.


The ODS SIOC Reference Guide describes how each application realm is mapped to SIOC alongside SPARQL Query examples for interacting with the SIOC instance data.

Sparql Endpoint

ODS SPARQL endpoints provide access to SIOC instance data from a range of ODS-Application instances

The Live ODS Demo Server and MyOpenLink.net (Alpha) Service are examples of ODS instances that expose SIOC instance data to SPARQL Query Service clients.
